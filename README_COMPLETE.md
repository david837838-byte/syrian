## 🧪 الاختبار السريع

### 1. اختبر الإيداع بدون محفظة:
```bash
curl -X POST http://localhost:5000/api/transactions/deposit \
  -H "Authorization: Bearer ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "currency": "USDT",
    "network": "TRC20",
    "amount": 100,
    "tx_hash": "0x123"
  }'
```
**المتوقع:** خطأ واضح مع طلب إضافة محفظة

### 2. أضف محفظة:
```bash
curl -X POST http://localhost:5000/api/admin/receiving-wallets \
  -H "Authorization: Bearer ADMIN_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "currency_id": 1,
    "network_id": 1,
    "address": "TWyJvH6u8NQ6L1m3MvUhBN1jVLQp5Fc2mG",
    "label": "محفظة USDT"
  }'
```
**المتوقع:** ✅ 201 Created

### 3. اختبر الإيداع مرة أخرى:
```bash
curl -X POST http://localhost:5000/api/transactions/deposit \
  -H "Authorization: Bearer USER_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "currency": "USDT",
    "network": "TRC20",
    "amount": 100,
    "tx_hash": "0x123"
  }'
```
**المتوقع:** ✅ 201 Created مع معرّف المعاملة

## ❓ الأسئلة الشائعة

### س: هل يمكن حذف محفظة بعد استخدامها؟
**ج:** نعم، لكن تأكد من عدم وجود معاملات معلقة

### س: كم عدد المحافظ التي يمكن إضافتها؟
**ج:** غير محدود، أضف ما تريد

### س: هل الأموال تنتقل تلقائياً؟
**ج:** لا، المدير يتحقق من المعاملة ويوافق عليها يدوياً

### س: ماذا لو أدخلت عنوان خاطئ؟
**ج:** احذف المحفظة وأضفها مرة أخرى بالعنوان الصحيح
