## 🚀 خطوات البدء السريع

### 1. التثبيت المحلي

```bash
# تثبيت المتطلبات
pip install -r requirements_new.txt

# إعداد ملف البيئة
cp .env.example .env

# تشغيل التطبيق
python app.py
```

### 2. مع Docker

```bash
# بناء وتشغيل
docker-compose -f docker-compose_improved.yml up -d

# عرض السجلات
docker-compose logs -f app

# إيقاف
docker-compose down
```

### 3. تشغيل الاختبارات

```bash
# تثبيت أدوات الاختبار
pip install pytest pytest-cov

# تشغيل الاختبارات
pytest test_app.py -v

# مع التغطية
pytest --cov test_app.py
```
