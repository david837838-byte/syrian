💥 PRIMARY PROBLEM
─────────────────────────────────────────────────────────────────────────────

Users must SELECT/HIGHLIGHT text to read it properly. Text colors are too weak
and don't meet WCAG accessibility standards.

COMPLAINT: "I have to select text with my mouse to read it - fonts are invisible"

═══════════════════════════════════════════════════════════════════════════════

🔍 TECHNICAL DETAILS
─────────────────────────────────────────────────────────────────────────────

ROOT CAUSE: Poor color contrast between text and background

Current problematic colors in modern-theme.css:

1. --text-soft: #9fb0c5
   • Used for: Secondary text, descriptions, helper text
   • Problem: 3:1 contrast ratio (FAILS WCAG AA standard of 4.5:1)
   • Status: ❌ UNREADABLE

2. --type-copy-muted: #496175
   • Used for: Muted text, secondary descriptions
   • Problem: Insufficient contrast in dark backgrounds
   • Status: ❌ HARD TO READ

3. --type-kicker-soft: #2bb693
   • Used for: Section badges, kickers, highlights
   • Problem: Not visible enough against dark backgrounds
   • Status: ⚠️ WEAK

4. Borders with excessive transparency:
   • --control-border: rgba(255, 255, 255, 0.1)
   • --glass-border: rgba(255, 255, 255, 0.08)
   • Problem: Almost invisible borders
   • Status: ❌ NOT VISIBLE

═══════════════════════════════════════════════════════════════════════════════

✅ PROPOSED SOLUTION
─────────────────────────────────────────────────────────────────────────────

Replace these CSS variables and styles:

COLOR CHANGES:
────────────────

OLD → NEW:

1. --text-soft: #9fb0c5 → #d4e4f7
   Reason: Increases contrast from 3:1 to ~6:1 (WCAG AAA compliant)
   Impact: All secondary text becomes readable

2. --type-copy-muted: #496175 → #b8cfe5
   Reason: Better visibility for muted descriptions
   Impact: Helper text and descriptions become clear

3. --type-kicker-soft: #2bb693 → #5fecc0
   Reason: Improved visibility for section highlights
   Impact: Badges and kickers are now visible

4. input::placeholder: (none) → #b8cfe5 with opacity: 1
   Reason: Placeholders were invisible
   Impact: Users can now see input hints

5. form-control text: var(--text-soft) → #f8fbff
   Reason: Input text color was too weak
   Impact: Form fields are now readable

6. .main-nav a: var(--text-soft) → #f8fbff
   Reason: Navigation links were barely visible
   Impact: Menu is now clear and navigable

7. --control-border: rgba(255, 255, 255, 0.1) → rgba(255, 255, 255, 0.15)
   Reason: Borders were almost invisible
   Impact: Form boundaries are now visible

8. --glass-border: rgba(255, 255, 255, 0.08) → rgba(255, 255, 255, 0.15)
   Reason: Glass effect borders too transparent
   Impact: UI elements are now distinguishable
