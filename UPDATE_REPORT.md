## 🧪 خطوات الاختبار

### 1. إضافة محفظة جديدة
```bash
POST /api/admin/receiving-wallets
{
    "currency_id": 1,
    "network_id": 1,
    "address": "TWyJvH6u8NQ6L1m3MvUhBN1jVLQp5Fc2mG",
    "label": "محفظة USDT الرئيسية"
}
```

**المتوقع:** ✅ 201 Created

### 2. جلب قائمة المحافظ
```bash
GET /api/admin/receiving-wallets
```

**المتوقع:** ✅ 200 OK مع قائمة المحافظ

### 3. حذف محفظة
```bash
DELETE /api/admin/receiving-wallets/1
```

**المتوقع:** ✅ 200 OK

### 4. محاولة إيداع بدون محفظة
```bash
POST /api/transactions/deposit
{
    "currency": "USDT",
    "network": "TRC20",
    "amount": 100,
    "tx_hash": "0x123"
}
```

**المتوقع:** ❌ 400 Bad Request مع رسالة محسّنة
